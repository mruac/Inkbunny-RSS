Inkbunny API 2 RSS

Converts Inkbunny Search API to a valid RSS feed

Refer to Inkbunny Search API under Search Condition Parameters for information on available flags.
Please note that keywords=yes and orderby=create_datetime will be on by default unless specified otherwise!

rating flag: Use 0 and 1 to toggle the filter off/on and the following order to specify your rating filters: [General][Mature][Adult]. See examples for, well, exmaples.

Examples
  • text=anthro&title=yes
    Returns search results for Text search for "anthro" in Keywords and Title.
  • username=Inkbunny
    Returns submissions by that user.
  • favs_user_id=00001
    Returns submissions favourited by that user.
  • pool_id=0001
    Returns submissions from that pool.

  • (no flags) Returns all posts
  • type=4
    Returns all comic type submissions
  • rating=110
    Shows only General and Mature rated submissions
  • rating=001
    Shows only Adult rated submissions
IDs can be retrieved from the URL by accessing the main site and copying the IDs from the link.
